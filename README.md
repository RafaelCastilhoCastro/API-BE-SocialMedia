# labook-39
## Repositório do projeto Labook com alguns recursos Backend de uma rede social.

- Arquitetura Limpa (Clean Architecture)
- POO (OOP)

### Dependencias
* Express
* Cors
* Knex
* MySQL
* Dotenv
* Typescript
* UUID


### Endpoints disponíveis
* Create new user
* Get all users
* Create new post
* Get post info
* Create friendship
* Delete friendship
* Get feed
